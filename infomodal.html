About this app

This is a submission for the USGS App Challenge

Data sources include: ITIS, Phylotastic, Global Invasive Species Database, Phylomatic, and GBIF"

Source code for this app available on Github. Created by rOpenSci with lots of help from Ramnath Vaidyanathan. Vist our website to explore our R packages and tutorials. Get the R packages: taxize, rgbif. This app was built using Shiny. We use Phylomatic to generate the phylogeny, so phylogenies are restricted to plants.
