Run the code locally

install.packages("taxize")
library(taxize)
species <- c("species1","species2","species3")
tax_name(query=species, get=c("genus", "family", "order", "kingdom"), db="ncbi", locally=FALSE)

Info

Data from the ITIS API

Data Fields
  • genus: the genus
  • family: the family
  • order: the order
  • class: the class
  • kingdom: the kingdom
